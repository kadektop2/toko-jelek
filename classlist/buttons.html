Buttons

There is a nice button for any situation
<a class="zui button" href="http://example.com/">Text</a>
<button class="zui button green big">Text</button>
<input type="reset" class="zui button red" value="Text" />

Defaults

This is the default size and color. Anything is optional, the only requirement to create a button is the class .button. Using this class will define the element as to be a button, other classes will change it's style(color, size, etc.).

default white red green blue orange yellow pink purple teal olive gray black primary button Button Button Link

No BG

What if you want to create buttons that looks like links? You can use .nobg class to preserve the background and border only for hover state, and by default to show only show the text with its respective color, nothing more. Here are a few examples:

default red green blue orange yellow pink purple teal olive gray black primary button nobg Button Button

Buttons size

In many different situations the default button size may not be enough. Changing the size is super easy and this require only an additional CSS class. Besides default size, a button can be mini,small, big and huge.

The difference between mini and small is that mini has the minimal possible height and padding and the font size is equal to the body, and the small is almost like default button but the font size is lower and the padding as well.

mini small default big huge

The buttons can have between 3 different shapes: default, square and rounded corners.

Buttons group

Buttons group is something similar to a toolbar. Button groups does not have any limits, you can combine different colors, size and shapes, but anyway try to not mix different sizes.
